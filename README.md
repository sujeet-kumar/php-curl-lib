# php-curl-lib
PHP Curl Library

-by Sujeet <sujeetkv90@gmail.com>

Install using composer:

```json
{
    "repositories": [
        {
            "type": "vcs",
            "url": "https://github.com/sujeetkv/php-curl-lib"
        }
    ],
    "require": {
        "sujeetkv/php-curl-lib": "dev-master"
    }
}
```
